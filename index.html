Правила шифра Виженера

Шифр Виженера — это метод симметричного шифрования, который использует последовательность букв ключа для шифрования текста.

Пример:

Предположим, у нас есть слово "ПРИВЕТ" и ключ "КЛЮЧ".

Мы повторяем ключ до длины слова: "КЛЮЧКЛ"

Находим в квадрате Виженера зашифрованный символ по следующему принципу: столбец исходной буквы, строка буквы ключа. На их пересечении и будет находиться нужный нам символ:

П + К = Ъ
Р + Л = Ь
И + Ю = Ж
В + Ч = Щ
Е + К = П
Т + Л = Ю

Результат: "Ъьжщпю"

В нашей игре используется ключ "КЛЮЧ". Попробуйте зашифровать свою фамилию!
